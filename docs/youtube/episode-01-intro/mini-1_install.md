# Mini-Episode 1.1: Installing Sona

> Get Sona running on your computer in 5 minutes

## Script

### Intro (0:00 - 0:15)
"Let's get Sona installed on your computer. This takes just a few minutes."

### Download (0:15 - 1:00)
"First, go to [website]. Click Download. Choose your operating system - Windows, Mac, or Linux."

### Install - Windows (1:00 - 2:00)
"On Windows:
1. Run the installer
2. Click Next, Next, Install
3. Check 'Add to PATH'
4. Done!"

### Install - Mac/Linux (2:00 - 3:00)
"On Mac or Linux:
```bash
# Using pip
pip install sona

# Or download the installer
```

### Verify Installation (3:00 - 4:00)
"Let's make sure it works. Open your terminal or command prompt."

```bash
sona --version
```

"You should see the version number. If you do, you're ready!"

### VS Code Extension (4:00 - 4:45)
"For the best experience, install VS Code and the Sona extension:
1. Open VS Code
2. Go to Extensions (Ctrl+Shift+X)
3. Search 'Sona'
4. Install!"

### Outro (4:45 - 5:00)
"That's it! Sona is installed. In the next mini-episode, we'll write our first program."
