# Sona 0.14.1 - Sona-Native Stdlib Foundation

Sona `0.14.1` moves the public standard library foundation toward
Sona-authored `.smod` modules and adds release hardening for Python artifacts.

## Highlights

- Sona-authored `.smod` foundation modules: `queue`, `stack`, `sort`,
  `search`, `statistics`, `matrix`, `graph`, and `permissions`.
- Sona-authored wrappers over private intrinsics: `hashing`, `random`, `uuid`,
  `secrets`, `password`, `jwt`, and `crypto`.
- Manifest metadata now records stdlib module source and stability.
- Private intrinsics and `native_bridge` remain hidden from normal user module
  imports and user-facing stdlib lists.
- Release hardening now validates wheel, sdist, clean installed-package smoke
  checks, archive contents, and SHA-256 artifact hashes.

## Compatibility

No parser, grammar, NativeBridge redesign, dependency addition, public function
removal, PyPI upload automation, or marketplace publishing automation is
included in this release.

`crypto.encrypt_simple` remains preview/legacy obfuscation only and is not
production encryption.
