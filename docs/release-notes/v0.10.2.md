# Sona v0.10.2 - Release Notes

**Release tag:** `v0.10.2`  
**Date:** 2026-02-14

Sona v0.10.2 is a blocker-closure release focused on parser/runtime correctness, stdlib parity, and release hygiene for packaging.

## Highlights

- Fixed `repeat while` and `repeat until` runtime execution.
- Fixed multiline string parsing and execution for triple-quoted strings.
- Added `.smod` support for `string.template(...)`.
- Fixed `test.only(...)` and `assert.throws(...)` interoperability with Sona functions.
- Fixed LSP outline symbol positioning for `func` and `def`.
- Added `scripts/release_gate.ps1` for repeatable local release checks.

## Validation

- Targeted blocker scenarios pass:
  - `test_repeat_variants.sona`
  - `test_trailing_commas.sona`
  - `test_multiline_strings.sona`
  - `test_stdlib_additions.sona`
  - `test_test_module_extras.sona`
- `python -m pytest -q tests -x` passes.
- `python tools/run_all_tests.py` now runs blocker tests cleanly and only depends on local interpreter wiring for environment-specific smoke checks.

## Notes

- Python package metadata and runtime version are aligned to `0.10.2`.
- VS Code extension packaging can be produced with `vsce package` from `vscode-extension/`.
