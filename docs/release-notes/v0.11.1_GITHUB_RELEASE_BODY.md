# Sona v0.11.1

Minor stabilization release for control-flow correctness and parser startup performance.

## Highlights

- Fixed `when` subject matching semantics.
- Removed dead `continue` path and ghost control-flow hooks.
- Added `for a, b in value` destructuring.
- Reused a single parser instance in the interpreter to avoid repeated Earley parser rebuilds.

## Scope Guard

- This release is intentionally small.
- It includes validated correctness and startup fixes only.
- Larger cognitive-runtime architecture work is being deferred to a later release line.

## Verification

- Dedicated v0.11.1 control-flow regression suite
- Runtime/version alignment to `0.11.1`

## Notes

- This release is intended to stay narrow and low-risk.
- Extension dependency advisories should be reviewed if extension artifacts are published with the release.
