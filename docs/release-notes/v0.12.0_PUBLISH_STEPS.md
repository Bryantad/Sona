```powershell

git add -A
git commit -m "Release v0.12.0"
git push origin main
git tag -a v0.12.0 -m "Sona v0.12.0"
```

1. Open `https://github.com/Bryantad/Sona/releases/new`.
2. Choose tag `v0.12.0`.
3. Target branch: `main`.
4. Release title: `Sona v0.12.0`.
5. Paste the contents of `docs/release-notes/v0.12.0_GITHUB_RELEASE_BODY.md` into the release description.
6. Attach these artifacts:
