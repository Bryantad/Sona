# Sona v0.10.1 — Release Notes

**Release tag:** `v0.10.1`  
**Date:** 2026-01-20

Sona v0.10.1 is a stability + alignment release: it consolidates the Cognitive Preview direction, improves documentation organization, and keeps packaging/version metadata consistent across the repo.

## Highlights

- Cognitive Preview foundation: intent markers, decision provenance scaffolding, and trace-friendly workflows.
- Report-oriented workflows: report export pathways and diagnostics hooks (runtime + LSP).
- Version alignment: Python package version and docs reference `0.10.1`.

## Install

### Python (runtime + CLI)

```powershell
# Windows
python -m venv .venv
.\.venv\Scripts\Activate.ps1
pip install -r requirements.txt
pip install -e .

python -c "from sona import __version__; print(__version__)"
```

### Run a file

```powershell
python run_sona.py demo.sona
```
