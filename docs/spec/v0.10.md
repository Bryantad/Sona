This document freezes the **v0.10.0** surface area as it exists in this repository.

- Version: **0.10.0**
- Status: **Cognitive Preview baseline**
- Goal: deterministic, offline-first runtime with a complete stdlib

## In Scope (v0.10.0)

### Core language

- Variables: `let`, `const`
- Functions: `func name(params) { ... }`
- Control flow: `if`, `while`, `for`, `repeat N { ... }`
- Exceptions: `try { } catch { } finally { }`, `throw`
- Modules:
  - `import module`
  - `import module as alias`
  - `.smod` module loading (legacy module format)

### Expressions

- Arithmetic, boolean operators, comparisons
- Member access: `math.PI`, `fs.exists(path)`
- Calls with:
  - keyword args (`fn(a=1)`)
  - spread args **in calls only** (`fn(...args)`)

### Cognitive preview

- Built-in helpers exposed in runtime (e.g. `focus`, `working_memory`, `intent`, `decision`, `cognitive_report`)
- Offline-safe fallback behavior when AI is unavailable

### Stdlib

- Stdlib is shipped and verified by `run_all_tests.ps1`.
- Canonical list is in `sona/stdlib/MANIFEST.json`.

## Explicitly Out of Scope (not supported in v0.10.0)

These may exist in planning docs, but are **not part of the locked v0.10.0 runtime**:

- Selective imports (`from x import y`) (planned)
- Spread in list/dict literals (`[...a]`, `{...a}`) (planned)
- Arrow functions (`x => x+1`) (planned)
- `repeat { ... } until expr` (planned)

## Compatibility Notes

- Module resolution prefers stdlib modules; project-local packages live under `.sona_modules/` (see `docs/packages/manifest.md`).
- Windows console uses UTF-8 in `run_sona.py`.
