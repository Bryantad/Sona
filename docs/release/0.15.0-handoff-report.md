## Scope Status

Implemented and stable:

- 21 stable accessibility modules.
- 11 stable utility module surfaces.
- Guardian MVP with SHA-256 inventory, trusted local config, snapshots, diff,
  quarantine, rollback, verification, circuit breaker, PARG graph, reports,
  and audit history.
- Static manifest-backed stdlib discovery and probes.

Implemented and experimental:

- 34 cognitive-accessibility preview modules.
- Experimental modules are local-only by default and require explicit
  `{ "experimental_accessibility": true }` before session-state, adaptive, or
  persistent behavior is enabled.

Interface-ready but deferred:

- Signed release manifests.
- External trust anchors.
- Key rotation.
- Provenance attestations.
- Secure update metadata.

Not implemented:

- Optional parser enhancements. No parser or grammar changes were made for
  `0.15.0`.
