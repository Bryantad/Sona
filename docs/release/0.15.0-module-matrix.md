# Sona 0.15.0 Module Matrix

This matrix is the release inventory for the cognitive-accessibility and
Guardian resilience surfaces. Experimental modules are implemented with narrow,
local-only runtime behavior. State-changing, adaptive, or session-history
behavior requires explicit local opt-in with:

```json
{
  "experimental_accessibility": true
}
```
