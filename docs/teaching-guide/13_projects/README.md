# Module 13: Building Projects

## Overview
Time to put it all together! This module guides you through building real applications from start to finish.

**Target Audience:** Ages 12-55, neurodivergent-friendly  
**Prerequisites:** Modules 01-12  
**Duration:** 90-120 minutes

---

## Learning Objectives
- Plan and structure a project
- Combine multiple modules effectively
- Debug and test your code
- Create user-friendly programs

## Project Ideas by Difficulty

### Beginner Projects
- Todo list with file storage
- Simple calculator
- Mad libs generator
- Number guessing game

### Intermediate Projects
- Contact manager
- Quiz application
- Weather app (API)
- Password generator

### Advanced Projects
- Chat application
- Task scheduler
- File organizer
- Data dashboard

## Quick Reference: Project Structure

```
my_project/
├── main.sona           # Entry point
├── config.sona         # Configuration
├── models/
│   ├── user.sona      # Data models
│   └── task.sona
├── services/
│   ├── storage.sona   # Business logic
│   └── api.sona
├── utils/
│   └── helpers.sona   # Utility functions
└── tests/
    └── test_main.sona # Tests
```
