### What is Sona?

**Sona is a programming language designed to be easy to read and write.**

Here's what Sona code looks like:

```sona
print("Hello, world!")
```

That's it! One line, and your computer will display "Hello, world!" on the screen.

### Why Sona?

| Feature | What It Means |
|---------|---------------|
| **Clean syntax** | Code looks like English sentences |
| **Forgiving** | Makes helpful suggestions when you make mistakes |
| **Neurodivergent-friendly** | Designed with clear, predictable patterns |
| **Real power** | Can build actual applications, not just toy examples |
