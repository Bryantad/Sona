## Project Ideas by Level

### Beginner Projects
1. **Personal Journal** - Save daily entries with dates
2. **Expense Tracker** - Track spending by category
3. **Flashcard App** - Study with spaced repetition
4. **Recipe Manager** - Store and search recipes
5. **Habit Tracker** - Track daily habits

### Intermediate Projects
1. **REST API Client** - Interact with public APIs
2. **Markdown Converter** - Convert markdown to HTML
3. **File Organizer** - Auto-organize downloads

3. **Static Site Generator** - Markdown to website
4. **Package Manager** - Dependency management
5. **Language Tool** - Linter or formatter for Sona

## Teaching Others

The best way to solidify knowledge: **teach it!**

### Ways to Teach
- Write blog posts
- Create YouTube tutorials
- Answer questions on forums
- Mentor beginners
- Give talks at meetups

### Teaching Tips
1. **Start simple** - Assume no prior knowledge
2. **Use examples** - Show, don't tell
3. **Be patient** - Everyone learns differently
4. **Encourage questions** - No question is dumb
5. **Celebrate progress** - Small wins matter

## Daily Coding Practice

### The 30-Minute Rule
Code for at least 30 minutes every day:

```
Week 1-2: Review basics, small exercises
Week 3-4: Build a small project
Week 5-6: Add features, refactor
Week 7-8: Start a new project
```

### Code Kata Ideas
1. Implement a data structure (Stack, Queue, Tree)
2. Solve an algorithm problem
3. Refactor old code
4. Add tests to existing code
5. Read and understand someone else's code
