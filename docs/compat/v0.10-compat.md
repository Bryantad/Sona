This is the contract a v0.10.0 module/program can assume.

## Runtime guarantees

- Deterministic execution (no randomness unless user calls `random.*`).
- Stable stdlib API surface as shipped in `sona/stdlib/MANIFEST.json`.
- Errors are raised as Python exceptions internally; user-facing messages may be simplified.

## Module resolution

Resolution order for `import name`:

1. Project-local installed modules in `.sona_modules/` (if present)
2. Stdlib `.py` modules in `sona/stdlib/`
3. Stdlib `.smod` modules (legacy)

Project-local `.sona_modules/` supports:

- `.sona_modules/name.smod`
- `.sona_modules/name/__init__.smod`

## Filesystem + encoding

- Source files are read as UTF-8 by default (BOM tolerated).

## Backwards compatibility intent

- v0.10.0 does not guarantee forward-compat syntax acceptance for planned v0.10.1 features.
