v0.10.0 is the _baseline_ for deterministic error reporting.

## What you can rely on

- Syntax errors stop execution.
- Runtime errors stop execution unless caught with `try/catch`.
- `run_sona.py` prints a human-readable message; `--debug` includes a Python traceback.

## Common runtime errors

- Undefined variable: `NameError`
- Invalid import: `ImportError`
- Invalid index/key: typically `IndexError` / `KeyError`
- Wrong argument count: `TypeError`

## Notes for v1.0

A unified error printer with `file:line:col` + error codes is a v1.0 readiness item.
