# Change Log

All notable changes to the "sona-language-support" extension will be documented in this file.

## [0.9.0] - 2025-08-17

### Added

- Initial release of Sona Language Support for VS Code
- Comprehensive AI-powered development features
- Cognitive accessibility support (Neurotypical, ADHD, Dyslexia modes)
- Real-time AI integration with Azure OpenAI
- Syntax highlighting and language configuration for .sona files
- Code generation, refactoring, explanation, and debugging commands
- Focus mode and cognitive memory features
- Interactive onboarding and profile selection
- Enterprise-grade AI routing with fallback providers
- Full syntax highlighting for Sona language constructs
- Cognitive and AI-specific keyword highlighting

### Features

- **AI Commands**: Generate, refactor, explain, debug, and optimize Sona code
- **Cognitive Support**: Working memory, focus modes, and cognitive load management
- **Accessibility**: Specialized UI modes for different cognitive needs
- **Language Support**: Full syntax highlighting and IntelliSense-ready configuration
- **Real-time Integration**: Direct connection to Sona AI infrastructure

### Technical

- TypeScript implementation with robust error handling
- Azure AI integration with enterprise routing
- Persistent cognitive memory across sessions
- Real-time performance monitoring
- Configurable timeout and retry mechanisms
